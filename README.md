## Project Overview
This is a console-based Student Management System developed using Python.  
It allows administrators to manage student records efficiently with persistent storage using CSV files.


## Features
- Secure login authentication
- Add new student records
- View all students in tabular format
- Update existing student details
- Delete student records
- Search students by name or department
- View top-performing students based on marks
- Automatic student ID generation
- Input validation and error handling


## Technologies Used
- Python 3
- CSV File Handling
- File Handling
- Exception Handling
- Datetime Module
